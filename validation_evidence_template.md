## Decision Framework

### GO Criteria (ALL must be true)

- [ ] 10+ sources (majority Tier 1)
- [ ] 3+ behavioral proofs
- [ ] Clear pattern identified
- [ ] No critical contradictions
- [ ] Problem is operational (not just preference)
- [ ] Evidence from 2025-2026 (recent)

### NO-GO Criteria (ANY is true)

- [ ] <5 sources total
- [ ] <2 behavioral proofs
- [ ] Contradictory evidence dominates
- [ ] Problem is validated but already well-solved
- [ ] No willingness to pay signals
- [ ] Evidence is mostly Tier 3

### ADJUST Criteria

- [ ] Partial validation - need more research
- [ ] Validated for segment A but not segment B
- [ ] Problem validated but solution direction unclear
- [ ] Need expert calls to resolve specific questions
